Ответы на вопросы

Можно ли мигрировать мою почту с другого почтового решения? Как происходит процесс миграции?
Да, можно. Подробное описание процесса миграции — в нашей Базе знаний: https://help.mail.ru/biz/domain/migration
Можно ли подключить на платформе кириллический домен?
К сожалению, платформа «Mail.ru для бизнеса» не поддерживает домены на кириллице.
Можно ли использовать сервисы бесплатно?
Да, облачные сервисы Почта для домена, Корпоративный мессенджер Myteam и Файловое хранилище Teambox имеют бесплатные базовые тарифы.
Какое количество пользователей на домене можно создать на бесплатном тарифе?
До 3000.
Как узнать стоимость решения для 300+ пользователей?
Отправьте заявку на info@mail.ru — мы рассчитаем.
Можно ли подключить Почту для домена к почтовому клиенту?
Да. Подробная инструкция по подключению: https://help.mail.ru/biz/domain/userguide
Можно ли использовать Почту для домена для массовых рассылок?
Нет. Для массовых рассылок следует использовать специализированные сервисы, например Sendbox от Mail.ru.
Есть ли лимиты на размер ящиков в почте и вложений?
Нет, в Почте для домена Mail.ru нет лимитов на размер ящиков и вложений в письма. Файлы тяжелее 25 МБ будут загружаться в Облако.Mail.ru и приходить в виде ссылки.
Есть ли лимиты по количеству отправленных писем в час (в день)?
Да. Такие лимиты индивидуальны и рассчитываются автоматически — в зависимости от многих факторов, в том числе репутации ящика. Мы не рекомендуем использовать Почту для домена для массовых рассылок — для них подойдут специализированные сервисы, например Sendbox от Mail.ru.
